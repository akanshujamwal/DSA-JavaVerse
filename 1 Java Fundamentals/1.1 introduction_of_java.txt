Introduction to Java
Definition: Java is a high-level, object-oriented, platform-independent programming language.

Key Features:
* Platform Independent: Write once, run anywhere (via JVM).
* Object-Oriented: Supports classes, objects, inheritance, polymorphism.
* Simple & Secure: Easy syntax, memory management, and security features.
* Robust & Multi-threaded: Handles exceptions, supports concurrent programming.
* High Performance: Just-In-Time compiler optimizes performance.
